# BRIEF DESCRIPTION
-----------------------------------------------------------------------------------------------------------------------------------
The current document contains aethetics features extracted from the videos of the dev-set used in the context of the MediaEval 2018
 Predicting Media Memorability Task.
 
-----------------------------------------------------------------------------------------------------------------------------------
# FEATURE
-----------------------------------------------------------------------------------------------------------------------------------
Aesthetic visual features [1]
	outputs: a collection of features used in the prediction of visual aesthetics, composed of color, texture and object based descriptors, aggregated at video level by median and mean methods.
	file format: text file
	feature: a single list of numbers on one line, separated by comma (dimension = 109)
	
We would like to thank Mihai Gabriel Constantin (https://scholar.google.com/citations?user=A1oerNkAAAAJ&hl=en) for extracting the
 features.
